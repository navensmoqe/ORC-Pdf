الحصول على مفتاح API (Mistral)

يعتمد هذا التطبيق على تقنية ذكاء اصطناعي متطورة من Mistral AI، ولتشغيله تحتاج إلى مفتاح خاص بك:

1

زيارة منصة Mistral

توجه إلى الموقع الرسمي لمنصة Mistral عبر الرابط التالي:

console.mistral.ai
2

إنشاء حساب

قم بالتسجيل باستخدام بريدك الإلكتروني أو حساب Google الخاص بك.

3

إنشاء مفتاح API

من القائمة الجانبية، اختر "API Keys" ثم اضغط على زر "Create New Key".

4

نسخ المفتاح وتفعيله

انسخ الكود الذي سيظهر لك، ثم اذهب إلى إعدادات التطبيق (أيقونة الترس) والصقه هناك.

ملاحظة: لا تشارك هذا المفتاح مع أي شخص، فهو خاص بحسابك فقط.

كيفية استخدام التطبيق

رفع الملفات

يمكنك سحب وإفلات الصور أو ملفات PDF في المربع المخصص، أو الضغط عليه لاختيار ملف من جهازك.

المسح عبر الهاتف

إذا كنت تستخدم الهاتف، اضغط على أيقونة الكاميرا لالتقاط صورة مباشرة للمستند، أو أيقونة الرفع لاختيار ملف PDF مخزن.

المعالجة والتعديل

بعد انتهاء المعالجة، سيظهر النص في المحرر. يمكنك تعديله، إضافة روابط، أو تنسيقه (عريض، مائل، إلخ).

الحفظ والطباعة

بمجرد الانتهاء، يمكنك تحميل النص كملف نصي (.txt) أو طباعته مباشرة عبر أيقونات الأدوات.
